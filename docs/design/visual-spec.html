Visual Spec — Battle Screen

Implementer-ready tokens and component specs for direction 2a ("Vertical Duel"), chosen from the M0 wireframe pass.

Scope note: M0's own plan deliberately ships Story 5's UI unstyled (Tailwind defaults, no art) — the point of the spike is to prove the combat math, not the look. Treat this spec as ready whenever you apply it: drop it in at M0 if you'd rather build styled from day one, or hold it for M1's "one real UI" milestone and keep M0 truly bare. Either way, these are the exact values so no visual decisions are left to the implementer.

Color palette

One dark base, one warm accent (gold), one danger (monster/damage). Same accent-hue-family rule as the rest of the site — don't add more hues.

Layout — Vertical Duel (2a)

  • Outer frame: max-width 1040px, centered; 3px solid var(--border-gold) outer border + a 1px inset outline 8px in for a double-border look; padding 28px; background radial-gradient(ellipse at 50% 0%, #2a1710 0%, #140b07 65%).
  • Stack order, top → bottom: title → monster panel → divider ("VS" + gold diamonds) → player panel (HP + prompt/input/countdown + keyboard).
  • Monster panel: border-radius: 8px, 2px solid var(--border-gold), background linear-gradient(180deg, #3b1220, #2a0d17), padding 16px 20px.
  • Player panel: same shape, background linear-gradient(180deg, #241f3d20, #1a1226) — i.e. same structure as the monster panel, gold family instead of red.
  • HP bar: 14px tall, 7px radius, 1px border in the side's color family, fill is a gradient from the darker to brighter shade of that family.
  • Prompt/input row: prompt+input stacked on the left (flex: 1), countdown block fixed at 64px wide to its right.

Keyboard — all three states

Full 5-row QWERTY per the implementation plan: number row (ends in Backspace), two letter rows (second ends in Enter), Shift row, space row. Base key: 34×32px, 4px radius, 1px solid var(--border-gold-dim), background rgba(237,225,195,.06). Only three states ever apply color — everything else stays the neutral base key above.

Assets checklist

  • Google Fonts: Cinzel (500, 700), EB Garamond (400, 400 italic), JetBrains Mono (400, 500, 700) — all free/open license, no local font files needed.
  • No image assets for M0/M1 — the ornate borders/dividers above are CSS gradients + a rotated div ("◆"), not SVG or bitmap art. Real monster art doesn't arrive until M6.
  • tokens.json in this folder mirrors the palette/type table above in a machine-readable form for wiring into config/combat.ts's sibling theme file or Tailwind config.
